cor Hexadecimal: Um número hexadecimal é uma maneira de representar cores usando números e letras. Este é o código de cor que o usuário insere. Ele está no formato hexadecimal, e o código pega esse valor e muda gradualmente a cor baseando-se nesse código.

Incremento: Este valor define a velocidade com que as cores mudam.

Intervalo: O usuário pode definir quanto quer que as cores mudem a cada passo. Se colocar 1, a cor vai mudar um pouco; se colocar 10, vai mudar mais rapidamente.
